# AI 配置页面 UI 原型

这是一个高保真 HTML/CSS/JS 原型，展示了 AI 供应商和模型配置的交互流程。

## 如何预览

直接在浏览器中打开 `index.html` 文件即可。
推荐使用 Chrome 或 Edge 浏览器以获得最佳体验。

## 包含的功能

1. **供应商列表首页**：
   - 展示已配置的供应商（OpenAI, Google 等）。
   - 顶部入口进入“默认模型配置”。
   - 右上角 `+` 号添加新供应商。

2. **添加供应商页**：
   - 选择 API 格式 (OpenAI/Claude/Google)。
   - 输入名称、Base URL、API Key。

3. **模型配置详情页**：
   - 启用/禁用该供应商。
   - 修改 API Key。
   - 管理该供应商下的具体模型（启用/禁用）。
   - 手动添加新模型 ID。

4. **默认偏好设置页**：
   - 设置默认对话模型。
   - 设置默认识图模型。
   - 设置默认生图模型。
